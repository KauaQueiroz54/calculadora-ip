A Calculadora IP é um projeto desenvolvido em Python puro, sem o uso de nenhuma biblioteca externa. Todo o funcionamento foi implementado utilizando lógica e operações matemáticas básicas.
O sistema é executado diretamente pelo terminal e realiza de forma precisa o cálculo e análise de endereços IPv4.

## Funcionalidades
1. Cálculo da máscara de sub-rede e prefixo CIDR.
2. Determinação dos endereços de rede e broadcast.
3. Exibição do intervalo de hosts válidos.
4. Identificação da classe do IP (A, B, C, D ou E).
5. Verificação se o endereço é público ou privado.
6. Execução simples e leve, totalmente via linha de comando.

## Destaques Técnicos
1. Desenvolvida sem uso de bibliotecas externas — apenas lógica condicional, binária e operações aritméticas.
2. Código estruturado e comentado para facilitar a compreensão e manutenção.
3. Projeto criado como parte da disciplina de Redes de Computadores, com foco em consolidar o entendimento sobre endereçamento IP e subnetting.
